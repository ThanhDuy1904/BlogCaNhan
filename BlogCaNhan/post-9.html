1 PHP là gì?

PHP (Hypertext Preprocessor) là ngôn ngữ lập trình server-side mã nguồn mở, được thiết kế đặc biệt cho web development. PHP code được thực thi trên server và trả về kết quả HTML cho browser.

Đặc điểm chính của PHP

  • Server-side: Code chạy trên server, không phải client
  • Open Source: Miễn phí và có cộng đồng lớn
  • Cross-platform: Chạy trên Windows, Linux, macOS
  • Database friendly: Tích hợp tốt với MySQL, PostgreSQL, MongoDB...
  • Easy to learn: Cú pháp đơn giản, dễ tiếp cận
  • Embedded in HTML: Có thể nhúng trực tiếp vào HTML

PHP hoạt động như thế nào?

1. User gửi request đến server (ví dụ: truy cập trang web)
2. Server nhận request và tìm file PHP
3. PHP interpreter thực thi code PHP
4. PHP tương tác với database (nếu cần)
5. PHP tạo ra HTML output
6. Server gửi HTML về browser
7. Browser hiển thị nội dung cho user

File PHP đầu tiên

<?php
// File: hello.php
echo "Hello, World!";
?>

<!-- Nhúng PHP trong HTML -->
<!DOCTYPE html>
<html>
<head>
    <title>My PHP Page</title>
</head>
<body>
    <h1><?php echo "Welcome to PHP!"; ?></h1>
    <p>Current time: <?php echo date('Y-m-d H:i:s'); ?></p>
</body>
</html>

2 Tại sao nên học PHP?

Dễ học cho người mới bắt đầu

PHP có cú pháp đơn giản, tương tự C, Java. Không cần kiến thức sâu về lập trình để bắt đầu tạo website động.

Phổ biến nhất cho Web Development

Hơn 77% websites trên Internet sử dụng PHP (theo W3Techs). WordPress, Facebook, Wikipedia đều được xây dựng bằng PHP.

Tích hợp tốt với Databases

PHP có built-in support cho nhiều databases, đặc biệt là MySQL. Rất dễ để tạo các ứng dụng CRUD (Create, Read, Update, Delete). 

<?php
// Kết nối MySQL rất đơn giản
$conn = new mysqli("localhost", "username", "password", "database");

// Query dữ liệu
$result = $conn->query("SELECT * FROM users");

// Hiển thị kết quả
while($row = $result->fetch_assoc()) {
    echo "Name: " . $row["name"] . "<br>";
}
?>

3 Lịch sử PHP

PHP/FI (1995)

PHP được tạo ra bởi Rasmus Lerdorf vào năm 1995. Ban đầu là "Personal Home Page Tools" - một tập hợp các CGI scripts để quản lý trang web cá nhân.

PHP 3 (1998)

Viết lại hoàn toàn bởi Andi Gutmans và Zeev Suraski. Đổi tên thành "PHP: Hypertext Preprocessor" (recursive acronym). Thêm extensibility và database support.

PHP 4 (2000)

Ra mắt Zend Engine 1.0, cải thiện performance đáng kể. Thêm sessions, output buffering.

PHP 5 (2004)

Bước ngoặt lớn với Zend Engine 2.0. Thêm OOP đầy đủ (classes, interfaces, abstract classes), exceptions, PDO (PHP Data Objects).

PHP 7 (2015)

Cải thiện performance gấp đôi PHP 5. Thêm type declarations, spaceship operator, null coalescing operator. 

<?php
// PHP 7 features
function sum(int $a, int $b): int {
    return $a + $b;
}

// Null coalescing
$username = $_GET['user'] ?? 'Guest';

// Spaceship operator
echo 1 <=> 2; // -1
?>

PHP 8 (2020)

JIT (Just-In-Time) compilation, Union types, Named arguments, Attributes, Match expression. 

<?php
// PHP 8 features
// Union types
function foo(int|float $number) { }

// Named arguments
function greet(string $name, string $greeting = 'Hello') {
    return "$greeting, $name!";
}
echo greet(name: 'Duy', greeting: 'Hi');

// Match expression
$result = match($status) {
    'active' => 'User is active',
    'inactive' => 'User is inactive',
    default => 'Unknown status'
};
?>

4 Cài đặt môi trường PHP

Cách 1: XAMPP (Recommend cho beginners)

XAMPP là package bao gồm Apache, MySQL, PHP và phpMyAdmin. Dễ cài đặt, chạy được trên Windows, Mac, Linux.

  1. Download XAMPP từ apachefriends.org
  2. Cài đặt và khởi động Apache và MySQL
  3. Tạo file PHP trong thư mục htdocs
  4. Truy cập localhost/yourfile.php

Cách 2: WAMP (Windows)

Tương tự XAMPP nhưng chỉ dành cho Windows. Interface thân thiện hơn.

Cách 3: MAMP (Mac)

Dành cho macOS. Dễ sử dụng với giao diện đơn giản.

Cách 4: Docker (Advanced)

# docker-compose.yml
version: '3'
services:
  php:
    image: php:8.2-apache
    ports:
      - "8000:80"
    volumes:
      - ./src:/var/www/html
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: root
      MYSQL_DATABASE: mydb

Kiểm tra cài đặt

Tạo file info.php:

<?php
phpinfo();
?>

Truy cập localhost/info.php để xem thông tin PHP.

Text Editor / IDE

  • Visual Studio Code: Free, extensions phong phú
  • PHPStorm: Professional IDE (trả phí)
  • Sublime Text: Lightweight, nhanh
  • Notepad++: Simple cho Windows

5 Cú pháp cơ bản PHP

Biến và Kiểu dữ liệu

<?php
// Variables - bắt đầu với $
$name = "Duy";
$age = 20;
$price = 99.99;
$isStudent = true;

// Kiểu dữ liệu
$string = "Hello";           // String
$integer = 42;               // Integer
$float = 3.14;              // Float
$boolean = true;            // Boolean
$array = [1, 2, 3];        // Array
$null = null;              // NULL

// Constants
define("SITE_NAME", "TD Blog");
const VERSION = "1.0";

// Hiển thị biến
echo $name;
print $age;
var_dump($price);  // Hiển thị type và value
?>

Toán tử

<?php
// Arithmetic
$sum = 5 + 3;
$diff = 5 - 3;
$product = 5 * 3;
$quotient = 5 / 3;
$remainder = 5 % 3;
$power = 5 ** 3;

// Comparison
$equal = (5 == "5");        // true (loose)
$identical = (5 === "5");   // false (strict)
$notEqual = (5 != 6);
$greater = (5 > 3);

// Logical
$and = (true && false);
$or = (true || false);
$not = !true;

// String concatenation
$fullName = $firstName . " " . $lastName;
?>

Cấu trúc điều khiển

<?php
// If-else
if ($age >= 18) {
    echo "Adult";
} elseif ($age >= 13) {
    echo "Teenager";
} else {
    echo "Child";
}

// Switch
switch ($day) {
    case "Monday":
        echo "Start of week";
        break;
    case "Friday":
        echo "TGIF!";
        break;
    default:
        echo "Regular day";
}

// Ternary
$status = ($age >= 18) ? "Adult" : "Minor";
?>

Vòng lặp

<?php
// For loop
for ($i = 0; $i < 5; $i++) {
    echo $i . " ";
}

// While loop
$count = 0;
while ($count < 5) {
    echo $count++;
}

// Foreach (cho arrays)
$colors = ["red", "green", "blue"];
foreach ($colors as $color) {
    echo $color . " ";
}

// Foreach với key-value
$person = ["name" => "Duy", "age" => 20];
foreach ($person as $key => $value) {
    echo "$key: $value\n";
}
?>

Functions

<?php
// Khai báo function
function greet($name) {
    return "Hello, $name!";
}

// Function với default parameter
function greetWithTitle($name, $title = "Mr.") {
    return "Hello, $title $name!";
}

// Function với type declarations
function sum(int $a, int $b): int {
    return $a + $b;
}

// Variable functions
$func = 'greet';
echo $func('Duy');  // Hello, Duy!

// Anonymous function
$multiply = function($x, $y) {
    return $x * $y;
};
echo $multiply(3, 4);
?>

Arrays

<?php
// Indexed array
$fruits = ["Apple", "Banana", "Orange"];
echo $fruits[0];  // Apple

// Associative array
$person = [
    "name" => "Duy",
    "age" => 20,
    "city" => "HCM"
];
echo $person["name"];

// Multidimensional array
$students = [
    ["name" => "Duy", "grade" => "A"],
    ["name" => "An", "grade" => "B"]
];

// Array functions
$numbers = [1, 2, 3, 4, 5];
echo count($numbers);          // 5
array_push($numbers, 6);       // Add element
$last = array_pop($numbers);   // Remove last
$sorted = sort($numbers);      // Sort array
?>

6 Ứng dụng của PHP

Web Applications

PHP là backbone của hàng triệu websites và web applications. Từ blogs đơn giản đến enterprise applications phức tạp.

  • Facebook - mạng xã hội lớn nhất thế giới
  • Wikipedia - bách khoa toàn thư trực tuyến
  • WordPress - 43% websites sử dụng
  • Slack - công cụ giao tiếp nhóm

Content Management Systems (CMS)

Quản lý nội dung website dễ dàng:

  • WordPress - Blogs, business websites
  • Drupal - Government, education sites
  • Joomla - Community portals

E-commerce Platforms

  • Magento - Enterprise e-commerce
  • WooCommerce - WordPress plugin
  • PrestaShop - Online stores
  • OpenCart - Simple shopping carts

RESTful APIs

Tạo APIs cho mobile apps và single-page applications:

<?php
// Simple REST API endpoint
header('Content-Type: application/json');

if ($_SERVER['REQUEST_METHOD'] === 'GET') {
    $users = [
        ['id' => 1, 'name' => 'Duy'],
        ['id' => 2, 'name' => 'An']
    ];
    echo json_encode($users);
}
?>

Form Handling

<?php
// Process form submission
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $name = $_POST['name'];
    $email = $_POST['email'];
    
    // Validate
    if (filter_var($email, FILTER_VALIDATE_EMAIL)) {
        // Save to database
        echo "Form submitted successfully!";
    }
}
?>

File Operations

<?php
// Read file
$content = file_get_contents('data.txt');

// Write file
file_put_contents('output.txt', 'Hello World');

// Upload file
if (isset($_FILES['file'])) {
    $file = $_FILES['file'];
    move_uploaded_file($file['tmp_name'], 'uploads/' . $file['name']);
}
?>

Session và Authentication

<?php
// Start session
session_start();

// Login
if ($username === 'admin' && $password === 'pass') {
    $_SESSION['user_id'] = 1;
    $_SESSION['username'] = $username;
}

// Check if logged in
if (isset($_SESSION['user_id'])) {
    echo "Welcome, " . $_SESSION['username'];
}

// Logout
session_destroy();
?>
