Collections Framework - Quản Lý Dữ Liệu Hiệu Quả

1 Collections Framework là gì?

Collections Framework là một kiến trúc thống nhất để biểu diễn và thao tác với các tập hợp đối tượng. Nó cung cấp các interface (List, Set, Map, Queue) và các class triển khai (ArrayList, HashSet, HashMap...) để lưu trữ và xử lý dữ liệu một cách hiệu quả.

Lợi ích của Collections Framework

  • Giảm nỗ lực lập trình: Cung cấp sẵn các cấu trúc dữ liệu và algorithms
  • Tăng tốc độ: Các implementation đã được tối ưu hóa
  • Tái sử dụng: Các interface và class có thể sử dụng lại
  • Tính nhất quán: Tất cả collections có API tương tự nhau

2 List Interface

List là ordered collection (sequence), cho phép duplicate elements. Bạn có thể truy cập elements theo index.

ArrayList - Mảng động

import java.util.ArrayList;
import java.util.List;

public class ArrayListExample {
    public static void main(String[] args) {
        // Khởi tạo ArrayList
        List fruits = new ArrayList<>();
        
        // Thêm phần tử
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");
        fruits.add("Apple"); // Cho phép duplicate
        
        // Truy cập theo index
        System.out.println("First: " + fruits.get(0));
        
        // Duyệt qua List
        for (String fruit : fruits) {
            System.out.println(fruit);
        }
        
        // Kiểm tra tồn tại
        if (fruits.contains("Banana")) {
            System.out.println("Found Banana!");
        }
        
        // Xóa phần tử
        fruits.remove("Apple"); // Xóa phần tử đầu tiên
        fruits.remove(0); // Xóa theo index
        
        // Kích thước
        System.out.println("Size: " + fruits.size());
    }
}

LinkedList - Danh sách liên kết

import java.util.LinkedList;

public class LinkedListExample {
    public static void main(String[] args) {
        LinkedList numbers = new LinkedList<>();
        
        // Thêm ở đầu và cuối
        numbers.addFirst(10);
        numbers.addLast(20);
        numbers.add(30); // Thêm vào cuối
        
        // Lấy phần tử đầu và cuối
        System.out.println("First: " + numbers.getFirst());
        System.out.println("Last: " + numbers.getLast());
        
        // Xóa đầu và cuối
        numbers.removeFirst();
        numbers.removeLast();
    }
}

Khi nào dùng ArrayList vs LinkedList?

  • ArrayList: Dùng khi cần truy cập ngẫu nhiên nhiều (get/set theo index)
  • LinkedList: Dùng khi thêm/xóa phần tử ở đầu/cuối thường xuyên

3 Set Interface

Set là collection không cho phép duplicate elements. Mỗi phần tử chỉ xuất hiện một lần.

HashSet - Không thứ tự

import java.util.HashSet;
import java.util.Set;

public class HashSetExample {
    public static void main(String[] args) {
        Set cities = new HashSet<>();
        
        // Thêm phần tử
        cities.add("Hanoi");
        cities.add("HCMC");
        cities.add("Danang");
        cities.add("Hanoi"); // Không thêm được - duplicate
        
        System.out.println("Size: " + cities.size()); // 3
        
        // Kiểm tra
        if (cities.contains("HCMC")) {
            System.out.println("Found HCMC");
        }
        
        // Duyệt - không đảm bảo thứ tự
        for (String city : cities) {
            System.out.println(city);
        }
    }
}

LinkedHashSet - Duy trì thứ tự chèn

import java.util.LinkedHashSet;

public class LinkedHashSetExample {
    public static void main(String[] args) {
        LinkedHashSet numbers = new LinkedHashSet<>();
        
        numbers.add(30);
        numbers.add(10);
        numbers.add(20);
        numbers.add(10); // Không thêm được
        
        // Duyệt - theo thứ tự chèn
        for (Integer num : numbers) {
            System.out.println(num); // 30, 10, 20
        }
    }
}

TreeSet - Tự động sắp xếp

import java.util.TreeSet;

public class TreeSetExample {
    public static void main(String[] args) {
        TreeSet numbers = new TreeSet<>();
        
        numbers.add(30);
        numbers.add(10);
        numbers.add(20);
        numbers.add(40);
        
        // Tự động sắp xếp tăng dần
        for (Integer num : numbers) {
            System.out.println(num); // 10, 20, 30, 40
        }
        
        // Các methods đặc biệt
        System.out.println("First: " + numbers.first());
        System.out.println("Last: " + numbers.last());
        System.out.println("Higher than 25: " + numbers.higher(25));
        System.out.println("Lower than 25: " + numbers.lower(25));
    }
}

4 Map Interface

Map lưu trữ dữ liệu theo cặp key-value. Mỗi key là duy nhất và map đến một value.

HashMap - Không thứ tự

import java.util.HashMap;
import java.util.Map;

public class HashMapExample {
    public static void main(String[] args) {
        Map ages = new HashMap<>();
        
        // Thêm cặp key-value
        ages.put("Alice", 25);
        ages.put("Bob", 30);
        ages.put("Charlie", 28);
        ages.put("Alice", 26); // Update value của key "Alice"
        
        // Lấy value theo key
        System.out.println("Alice's age: " + ages.get("Alice"));
        
        // Kiểm tra key tồn tại
        if (ages.containsKey("Bob")) {
            System.out.println("Bob exists");
        }
        
        // Kiểm tra value tồn tại
        if (ages.containsValue(30)) {
            System.out.println("Someone is 30");
        }
        
        // Duyệt qua Map
        for (Map.Entry entry : ages.entrySet()) {
            System.out.println(entry.getKey() + ": " + entry.getValue());
        }
        
        // Duyệt chỉ keys
        for (String name : ages.keySet()) {
            System.out.println(name);
        }
        
        // Duyệt chỉ values
        for (Integer age : ages.values()) {
            System.out.println(age);
        }
        
        // Xóa
        ages.remove("Charlie");
        
        // getOrDefault
        int age = ages.getOrDefault("David", 0);
    }
}

LinkedHashMap - Duy trì thứ tự chèn

import java.util.LinkedHashMap;

LinkedHashMap map = new LinkedHashMap<>();
map.put("A", "Apple");
map.put("B", "Banana");
map.put("C", "Cherry");

// Duyệt theo thứ tự chèn
for (Map.Entry entry : map.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}

TreeMap - Tự động sắp xếp theo key

import java.util.TreeMap;

TreeMap map = new TreeMap<>();
map.put(3, "Three");
map.put(1, "One");
map.put(2, "Two");

// Tự động sắp xếp theo key
for (Map.Entry entry : map.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}
// Output: 1=One, 2=Two, 3=Three

5 Queue Interface

Queue là collection theo nguyên tắc FIFO (First In First Out) - phần tử vào trước sẽ ra trước. 

import java.util.LinkedList;
import java.util.Queue;

public class QueueExample {
    public static void main(String[] args) {
        Queue queue = new LinkedList<>();
        
        // Thêm phần tử vào cuối queue
        queue.offer("First");
        queue.offer("Second");
        queue.offer("Third");
        
        // Xem phần tử đầu tiên (không xóa)
        System.out.println("Peek: " + queue.peek());
        
        // Lấy và xóa phần tử đầu tiên
        System.out.println("Poll: " + queue.poll());
        System.out.println("Poll: " + queue.poll());
        
        // Kiểm tra rỗng
        if (!queue.isEmpty()) {
            System.out.println("Queue not empty");
        }
    }
}

PriorityQueue - Hàng đợi ưu tiên

import java.util.PriorityQueue;

PriorityQueue pq = new PriorityQueue<>();

pq.offer(30);
pq.offer(10);
pq.offer(20);

// Lấy theo thứ tự ưu tiên (nhỏ nhất trước)
System.out.println(pq.poll()); // 10
System.out.println(pq.poll()); // 20
System.out.println(pq.poll()); // 30

HashSet Không Không Không cần thứ tự, cần tốc độ
