1 JavaScript là gì?

JavaScript là một ngôn ngữ lập trình động, high-level, được sử dụng chủ yếu để tạo các trang web tương tác và động. Ban đầu chỉ chạy trên browser (client-side), nhưng với sự ra đời của Node.js, JavaScript giờ đây có thể chạy ở mọi nơi (server-side, mobile, desktop, IoT...).

Đặc điểm chính của JavaScript

  • Interpreted Language: Code được thực thi trực tiếp, không cần compile
  • Dynamically Typed: Không cần khai báo kiểu dữ liệu
  • First-class Functions: Functions là objects, có thể gán vào biến
  • Prototype-based OOP: Kế thừa thông qua prototypes
  • Event-driven: Phản hồi các events từ người dùng
  • Single-threaded: Một luồng thực thi nhưng có async capabilities

JavaScript vs Java

Mặc dù tên gọi giống nhau, JavaScript và Java là hai ngôn ngữ hoàn toàn khác biệt!

JavaScript Java
Interpreted Compiled
Dynamic typing Static typing
Chạy trên browser/Node.js Chạy trên JVM
Prototype-based OOP Class-based OOP

2 Tại sao nên học JavaScript?

Chạy trên mọi browser

JavaScript được hỗ trợ native bởi tất cả các trình duyệt hiện đại. Không cần cài đặt gì thêm, chỉ cần mở browser và bắt đầu code!

Dễ học cho người mới bắt đầu

Cú pháp đơn giản, không cần setup phức tạp. Bạn có thể thấy kết quả ngay lập tức trong browser's console. 

// Đơn giản thế này thôi!
console.log("Hello, JavaScript!");

// Tạo biến không cần khai báo kiểu
let name = "Duy";
let age = 20;

// Function đơn giản
function greet(name) {
    return "Hello, " + name + "!";
}

console.log(greet("World")); // Hello, World!

Đa năng - Full-stack Development

  • Frontend: React, Vue, Angular - Tạo UI tương tác
  • Backend: Node.js, Express.js - Xây dựng API, server
  • Mobile: React Native, Ionic - Phát triển app iOS/Android
  • Desktop: Electron - Tạo ứng dụng desktop (VS Code, Discord...)
  • Game Development: Phaser, Three.js
  • Machine Learning: TensorFlow.js

Ecosystem phong phú

NPM (Node Package Manager) có hàng triệu packages miễn phí. Bất cứ tính năng nào bạn cần, chắc chắn đã có ai đó làm rồi!

Cộng đồng lớn nhất

JavaScript là ngôn ngữ phổ biến nhất theo GitHub, Stack Overflow. Dễ dàng tìm tài liệu, tutorials, và sự hỗ trợ.

Cơ hội việc làm cao

Nhu cầu JavaScript developers luôn cao. Từ startups đến các tập đoàn lớn đều cần JavaScript cho web và mobile apps.

3 Lịch sử JavaScript

Nguồn gốc (1995)

JavaScript được tạo ra bởi Brendan Eich tại Netscape chỉ trong 10 ngày! Ban đầu có tên Mocha, sau đó là LiveScript, và cuối cùng đổi thành JavaScript để tận dụng độ phổ biến của Java (mặc dù hai ngôn ngữ không liên quan gì nhau).

Chiến tranh browsers (1996-2000)

Netscape và Microsoft cạnh tranh khốc liệt. Microsoft tạo ra JScript (phiên bản riêng của JavaScript) cho Internet Explorer. Điều này dẫn đến sự không tương thích giữa các browsers.

Chuẩn hóa - ECMAScript (1997)

ECMA International tạo ra chuẩn ECMAScript để thống nhất JavaScript. JavaScript là implementation của ECMAScript standard.

AJAX và Web 2.0 (2005)

Google Maps và Gmail cho thấy sức mạnh của AJAX (Asynchronous JavaScript and XML), mở ra kỷ nguyên web applications tương tác.

Node.js (2009)

Ryan Dahl tạo ra Node.js, cho phép JavaScript chạy trên server. Đây là bước ngoặt lớn, biến JavaScript từ ngôn ngữ chỉ chạy trên browser thành full-stack language.

Frameworks Era (2010s)

Sự ra đời của Angular (2010), React (2013), Vue (2014) đã thay đổi cách phát triển web apps.

4 Các phiên bản ECMAScript

ES5 (2009)

Phiên bản ổn định, được hỗ trợ rộng rãi. Thêm strict mode, JSON support, array methods.

// ES5 features
"use strict";

var numbers = [1, 2, 3, 4, 5];

// Array methods
var doubled = numbers.map(function(n) {
    return n * 2;
});

var evens = numbers.filter(function(n) {
    return n % 2 === 0;
});

ES6/ES2015 - Bước ngoặt lớn

Phiên bản quan trọng nhất với nhiều tính năng mới:

  • let và const: Block-scoped variables
  • Arrow functions: Cú pháp ngắn gọn cho functions
  • Classes: Syntactic sugar cho prototype-based OOP
  • Template literals: String interpolation
  • Destructuring: Trích xuất dữ liệu từ arrays/objects
  • Promises: Xử lý async tốt hơn callbacks
  • Modules: import/export
// ES6 features
const numbers = [1, 2, 3, 4, 5];

// Arrow function
const doubled = numbers.map(n => n * 2);

// Template literal
const name = "Duy";
const greeting = `Hello, ${name}!`;

// Destructuring
const person = { name: "Duy", age: 20 };
const { name, age } = person;

// Classes
class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    greet() {
        return `Hello, I'm ${this.name}`;
    }
}

ES2016 - ES2023

Mỗi năm có một phiên bản mới với các tính năng cải tiến:

  • ES2016: Array.includes(), Exponentiation operator (**)
  • ES2017: async/await, Object.entries(), Object.values()
  • ES2018: Rest/Spread properties, Async iteration
  • ES2020: Optional chaining (?.), Nullish coalescing (??)
  • ES2021: String.replaceAll(), Promise.any()

5 Ứng dụng của JavaScript

Frontend Web Development

Tạo giao diện người dùng tương tác, dynamic websites:

  • React: Library phổ biến nhất (Facebook, Instagram, Netflix)
  • Vue: Progressive framework, dễ học (Alibaba, Xiaomi)
  • Angular: Full framework từ Google

Backend Development

Xây dựng server, REST APIs, real-time applications:

  • Node.js: JavaScript runtime cho server
  • Express.js: Web framework phổ biến
  • NestJS: Framework enterprise với TypeScript

Mobile App Development

  • React Native: Phát triển iOS/Android (Facebook, Instagram)
  • Ionic: Hybrid mobile apps
  • NativeScript: Native mobile apps

Desktop Applications

  • Electron: VS Code, Discord, Slack, Skype
  • NW.js: Tương tự Electron

Game Development

  • Phaser: 2D game framework
  • Three.js: 3D graphics với WebGL
  • Babylon.js: 3D game engine

Data Visualization

  • D3.js: Data-driven documents
  • Chart.js: Simple charts
  • Plotly: Scientific charts

6 Cách bắt đầu học JavaScript

Bước 1: Học HTML và CSS trước (nếu chưa biết)

JavaScript thường được sử dụng cùng HTML/CSS để tạo websites. Nên học HTML/CSS cơ bản trước khi bắt đầu JavaScript.

Bước 2: Setup môi trường

Bạn chỉ cần:

  • Browser: Chrome, Firefox, Edge (đều có Developer Tools)
  • Text Editor: VS Code (recommend), Sublime Text, Atom

Không cần cài đặt gì thêm! Mở browser console (F12) và bắt đầu code:

// Trong browser console, gõ:
console.log("Hello, JavaScript!");

// Tạo biến
let message = "I'm learning JavaScript!";
console.log(message);

// Tính toán
let sum = 5 + 10;
console.log("5 + 10 =", sum);

Bước 3: Nắm vững cú pháp cơ bản

  • Variables: let, const, var
  • Data types: String, Number, Boolean, Array, Object
  • Operators: +, -, *, /, %, ==, ===, !=, !==
  • Control flow: if-else, switch, loops
  • Functions: declaration, expression, arrow functions

Bước 4: Học DOM Manipulation

Cách JavaScript tương tác với HTML:

// Select elements
const button = document.querySelector('#myButton');
const heading = document.querySelector('h1');

// Change content
heading.textContent = 'New Heading';

// Add event listener
button.addEventListener('click', function() {
    alert('Button clicked!');
});

Bước 5: Thực hành với projects nhỏ

  • Calculator
  • Todo List
  • Quiz App
  • Weather App (sử dụng API)

Bước 6: Học ES6+ features

Arrow functions, destructuring, spread operator, async/await...

Bước 7: Học một framework

Sau khi vững JavaScript thuần (vanilla JS), học React, Vue hoặc Angular.
